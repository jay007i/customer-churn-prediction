# Customer Churn Prediction

This project uses Machine Learning techniques to predict whether a customer is likely to leave a company (churn) or remain a customer. The project includes data cleaning, exploratory data analysis (EDA), feature engineering, visualization, model training, and evaluation.

## Objectives

* Identify customers at risk of churning
* Understand factors that influence customer retention
* Build predictive machine learning models
* Help businesses reduce customer loss and increase revenue

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Models

* Logistic Regression
* Decision Tree
* Random Forest

## Results

The model predicts customer churn based on customer demographics, account information, and service usage patterns.
